Privilege Escalation Matt -> root

Initial research revealed that MiniServ 1.910 was running on Port 10000. And that process is running with root privileges. If this version of MiniServ is vulnerable, it could be used to gain root privileges.

searchsploit reveals that this version is still vulnerable.

kali@kali:~$ searchsploit Webmin 1.910
